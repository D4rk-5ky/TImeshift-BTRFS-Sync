# Versioning note

This project was originally numbered too aggressively while the design was still
changing quickly. From this build onward, artifact zip versions are counted as
small patch-like steps:

```text
1st zip  -> 0.0.1
2nd zip  -> 0.0.2
3rd zip  -> 0.0.3
4th zip  -> 0.0.4
5th zip  -> 0.0.5
6th zip  -> 0.0.6
7th zip  -> 0.0.7
8th zip  -> 0.0.8
9th zip  -> 0.0.9
10th zip -> 0.1.0
11th zip -> 0.1.1
```

Based on the zip artifacts created in this conversation, this corrected build is
the 9th zip, so its version is `0.0.9`.

The previous `0.4.0` label should be treated as an over-large experimental
label, not as the real project maturity level.
